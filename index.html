KanZen — Quick start

Kanban without the noise. Single file, your folder, your data. No server, no account, no board limits.

Storage

Click 📁 to connect a folder. KanZen will write each board as a plain .kanzen.json file you can read, back up, sync with Dropbox/iCloud/git, or open in any text editor. If you don't connect a folder, boards live in this browser's IndexedDB and you can still export them as JSON.

Keyboard shortcuts

NNew card in the focused column (or first column)
E / EnterEdit focused card
/Focus search
← → ↑ ↓Move card focus
EscClose modals

Tips

  • Drag cards between columns and reorder within a column.
  • Drag column headers to reorder columns.
  • Filter cards by label / priority / due / member — non-matching cards dim instead of hiding, so you keep the board's shape.
  • Auto-save runs every 5 seconds when you make changes; destructive actions save immediately.
  • Browser-stored boards are exportable to JSON anytime.
