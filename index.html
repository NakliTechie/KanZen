KanZen — what it does

Kanban without the noise. Single file, your folder, your data. No server, no account, no board limits.

Header buttons

📁Connect a folder. Each board is then saved as a .kanzen.json file you can read, back up, or sync with Dropbox/iCloud/git. Without a folder, boards live in this browser's IndexedDB and you can still export them.
📸Snapshots — manual or auto (before destructive actions, optional daily). Pick two snapshots to compare them: card-level diff with added / removed / modified.
📜Activity feed — every change to the board, with timestamps.
🔗Share board via URL. The whole board is encoded into the URL hash fragment (compressed, optionally AES-GCM-encrypted with a passphrase). Nothing is uploaded — the recipient's browser decodes it locally. QR code is available for in-person sharing.
⚙Board settings — name, columns + WIP limits, members, labels, background (8 gradients, 8 solids, custom colour).
⇅Import / export — KanZen JSON (single board or all boards), CSV, Markdown, or import from Trello (lists, cards, labels, members, checklists).
↶ ↷Undo / redo — 50 deep per board. Cmd/Ctrl+Z / Shift+Z also work.
⤓Install as an app (only appears when your browser supports the install prompt).
◐Toggle dark / light theme.

Views

Switch between Board (kanban), List (sortable table — click any column header to sort), and Calendar (month grid — drag a card between days to set its due date) using the pills on the right of the filter bar. The view sticks per board.

Cards

Click any card to edit. You get a markdown description (with live preview), checklist with progress bar, due date, priority, labels, members, file attachments (embedded in the board JSON), and threaded comments. Comment counts and checklist progress show on the card face.

Filters

Filter by label, priority, due-date bucket (overdue / today / this week / no date), or member. Non-matching cards dim instead of hiding, so you keep the board's shape and your WIP limits stay meaningful.

Storage details

Auto-save runs every 5 seconds when there are changes; destructive actions (delete card / column / board) save immediately. The folder on disk is the source of truth — drop it in any sync tool (Dropbox, iCloud, Syncthing, git) and you have free multi-device sync. Each board is one human-readable JSON file you can grep, diff, or open in any text editor.
